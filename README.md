## 확장 설치하기(웬만하면 chrome)

- 누르고싶게 생긴 초록색 `</>Code` 클릭  
- `Download ZIP` 클릭 후 압축 풀기  
- Chrome 오른쪽 상단에 있는 점 세 개 클릭  
- **확장 프로그램 > 확장 프로그램 관리**  
- 오른쪽 상단에 있는 **개발자 모드 ON**  
- 화면 왼쪽 상단 "`압축해제된 확장 프로그램을 로드합니다`" 클릭  
- 압축 푼 폴더 선택

## 사용법

- Steam 프로필 → **프로필 수정** → **전시중인 전시대**  
- 화면 오른쪽 상단 퍼즐 아이콘 클릭  
- `steamachievement`로 시작하는 확장 클릭  
- 버튼 클릭  
- `F5` 눌러서 제대로 적용됐는지 확인
